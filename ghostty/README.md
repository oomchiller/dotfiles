# Ghostty

## Installation

```sh
cd ~/.config/dotfiles
stow ghostty
```

## Requirements:

- font: `JetBrainsMono Nerd Font Mono`
- shell: `/bin/zsh`

## Hotkeys

- `Cmd+A`       — send `Ctrl+B` to `tmux`
- `Cmd+Shift+C` — copy terminal selection to the system clipboard
- `Cmd+V`       — paste from the system clipboard
- `Cmd+W`       — close the current tab
