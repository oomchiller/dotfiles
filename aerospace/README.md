# Aerospace

## Installation

```sh
cd ~/.config/dotfiles
stow -t ~ aerospace
```

This package manages `~/.aerospace.toml` directly.

## Basics

- starts at login
- keeps 18 persistent workspaces
- pins workspaces `1-9` to monitor `1`
- pins workspaces `10-18` to monitor `2`
- uses `Alt+H/J/K/L` for focus and `Alt+Shift+H/J/K/L` for moves
- opens Ghostty with `Alt+Enter`
