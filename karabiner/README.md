# Karabiner

## Installation

```sh
cd ~/.config/dotfiles
stow -t ~ karabiner
```

This package manages `~/.config/karabiner/karabiner.json`.

## Local State

- `~/.config/karabiner/automatic_backups` is local state and is not stowed
- custom rules can also live in `~/.config/karabiner/assets/complex_modifications`
